# GeminiClaw Phase 1 Scaffold

This package contains a practical phase 1 scaffold for:

- Flutter desktop/web UI shell for AI Team OS
- FastAPI backend endpoint skeleton
- Realtime websocket event bridge
- Approval, task, logs, channels, and knowledge placeholders

## Directories

- `openclaw_app/` : Flutter client scaffold
- `openclaw_backend/` : FastAPI backend scaffold
