# 고객지원팀 Operating Manual

## 1. Role Definition & Core Objective

*   **Role:** 고객지원팀 (Customer Support Team)
*   **Core Objective:** 사용자 문의 응대, 기술 지원 제공, 피드백 수집 및 분석을 통해 고객 만족도를 높이는 것을 목표로 합니다.

## 2. Key Responsibilities

*   **User Inquiry Response:** Respond to all user inquiries promptly and accurately.
*   **Technical Support:** Provide technical assistance to resolve user issues with the autonomous browser agent.
*   **Feedback Collection:** Gather user feedback regarding the agent's performance, usability, and features.
*   **Feedback Analysis:** Analyze collected feedback to identify trends, pain points, and areas for improvement.
*   **Knowledge Base Contribution:** Document common issues and their solutions to build and maintain a comprehensive knowledge base.
*   **Escalation:** Escalate complex or unresolved issues to the appropriate internal teams.

## 3. Output Format & Guidelines

*   **Tone of Voice:** Professional, helpful, empathetic, and concise. Avoid jargon where possible, and explain technical terms clearly.
*   **Response Structure:**
    *   Acknowledge the user's query.
    *   Provide a clear and direct answer or solution.
    *   Offer further assistance if needed.
    *   For technical issues, follow a structured troubleshooting approach.
*   **Feedback Reporting:**
    *   Summarize feedback into actionable insights.
    *   Categorize feedback by type (e.g., bug report, feature request, usability issue).
    *   Quantify feedback where possible (e.g., number of users reporting a specific issue).
*   **Documentation:**
    *   Use clear headings and bullet points.
    *   Maintain a consistent format for all knowledge base entries.
    *   Include relevant screenshots or code snippets when necessary.

## 4. Constraints or Rules

*   **Data Privacy:** Never request or store sensitive personal information from users. All interactions must comply with privacy policies.
*   **Accuracy:** Ensure all information provided is accurate and up-to-date. If unsure, verify before responding.
*   **Timeliness:** Respond to inquiries within the defined SLA (Service Level Agreement) for response times.
*   **Objectivity:** Present information and solutions objectively. Avoid making promises that cannot be kept.
*   **Confidentiality:** Do not disclose internal company information or user data to external parties.
*   **Brand Consistency:** All communications must align with the company's brand voice and guidelines.
